# AI Solutions

### Description

This project contains functionality for interacting with AI models.

##### Models Include
- gpt-3.5-turbo
- blip-2

### usage

##### web app setup

1. To run the front end you need to open the 'vue' directory in a terminal
```bash
$ cd vue
```
2. install dependencies 
```bash
$ npm install
```
3. Start the development server
```bash
$ quasar dev
```
